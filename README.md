
# 💸 Trust Tracker

**Trust Tracke** is a modern expense tracker built with **React** and powered by **Supabase**. It helps you stay on top of your finances while offering more than just tracking — it's your smart assistant for budgeting, saving, and understanding where your money goes.

---

## 🚀 Features

- ✅ **User Authentication** (via Supabase)
- 💰 **Track Income & Expenses**
- 📊 **Visual Reports** (Pie/Bar charts for category-wise spending)
- 💡 **Smart Insights** (automated monthly summaries, trends)
- 📆 **Recurring Transactions** support
- 🔔 **Reminders** & budget alerts
- 🧾 **Custom Categories** & notes for every entry
- 🔍 **Filter & Search** transactions
- 🌙 **Dark Mode** support
- 📱 **Responsive UI** (mobile-first design)

## 📦 Installation

1. **Clone the repo**
   ```bash
   git clone https://github.com/kachakaran6/trust-tracker.git
   cd trust-tracker


2. Install dependencies

   bash
   npm install
   

3. Set up environment variables

   Create a `.env` file:

   
   VITE_SUPABASE_URL=your-supabase-url
   VITE_SUPABASE_ANON_KEY=your-anon-key
   

4. Start the app

   bash
   npm run dev

* All data is securely stored using Supabase's RLS (Row Level Security).
* Authenticated routes protected with Supabase JWT-based sessions.
